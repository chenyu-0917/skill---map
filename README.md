# 🗺️ 個人技術技能地圖

一個互動式的技術技能視覺化工具,用於追蹤個人技術成長與職涯發展規劃。

## ✨ 功能特色

- 📊 **統計儀表板** - 即時顯示技能總覽與成長指標
- 🎯 **雷達圖分析** - 視覺化呈現五大技術領域的能力分布
- 💪 **詳細技能清單** - 分類展示所有技能與熟練度
- 🚀 **成長路徑規劃** - 短中長期目標與學習建議
- 🎨 **現代化設計** - 深色主題、流暢動畫、響應式介面

## 🚀 快速開始

### 線上預覽

訪問 GitHub Pages: `https://你的使用者名稱.github.io/skill-map/`

### 本地運行

1. Clone 專案:
```bash
git clone https://github.com/你的使用者名稱/skill-map.git
cd skill-map
```

2. 直接在瀏覽器中開啟 `skill-map.html`

就這麼簡單!不需要安裝任何依賴套件。

## 📝 自訂技能資料

編輯 `skill-map.html` 中的 `skillsData` 物件來更新你的技能:

```javascript
const skillsData = {
    "後端開發": [
        { 
            name: ".NET Framework/Core", 
            level: "advanced", 
            score: 75, 
            description: "熟悉 ASP.NET Core、Web API 開發" 
        },
        // 新增更多技能...
    ],
    // 新增更多類別...
};
```

### 技能等級說明

- `beginner` (初學) - 0-40分
- `intermediate` (中階) - 41-65分
- `advanced` (進階) - 66-85分
- `expert` (專家) - 86-100分

## 🎨 自訂外觀

修改 CSS 變數來改變配色:

```css
:root {
    --primary: #2D5BFF;        /* 主要顏色 */
    --secondary: #00D9B5;      /* 次要顏色 */
    --accent: #FF6B6B;         /* 強調顏色 */
    /* 更多顏色設定... */
}
```

## 📦 技術架構

- **純前端** - 無需後端伺服器
- **Chart.js** - 雷達圖視覺化
- **原生 JavaScript** - 無框架依賴
- **CSS3 動畫** - 流暢的使用者體驗
- **響應式設計** - 支援各種裝置

## 🌟 成長路徑範例

專案包含三階段成長路徑規劃:

1. **短期目標 (3-6個月)** - 強化核心技術
2. **中期目標 (6-12個月)** - 拓展技能廣度
3. **長期目標 (12-24個月)** - 技術領導力發展

你可以根據自己的職涯目標調整 `roadmapData` 陣列。
